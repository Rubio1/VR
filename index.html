Esta página muestra información respecto al proyecto en desarrollo llamado Visualizador de Reportes Gráficos.

¿Qué es?

Visualizador de Reportes Gráficos es una aplicación para dispositivos móviles que permite observar información extraída de las bases de datos en tiempo real en forma de reportes gráficos, mismos que permite la interpretación de datos para observar analizar el comportamiento y rendimiento de los negocios.

¿Para qué se creó?

Para demostrar el amplio alcance que este conjunto de tecnologías pueden llegar a ofrecer para la visualización de reportes, ya que no está limitado a observar el rendimiento de los negocios, si no, analizar además el comportamiento de diversas áreas o sectores ya sea públicos o privados donde se requiera realizar algún monitoreo, análisis o consulta de datos.

Objetivo

Poner al alcance de la mano la visualización de reportes gráficos con información en tiempo real, sin importar la hora ni el lugar donde se requiera hacer una consulta..

Estrategias

Mediante el uso de dispositivos móviles equipados con Android 2.1 o superior, tecnologías que auxilian el proceso de renderización de gráficas y conexión a Internet para obtener datos.
