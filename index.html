Bienvenidos a la pagina de VR.

Esta pagina muestra informacion respecto al proyecto en desarrollo llamado Visualizador de Reportes Graficos.

¿Que es?

Visualizador de Reportes Gráficos es una aplicación para dispositivos móviles que permite observar informacion extraida de las bases de datos en tiempo real en forma de reportes graficos, mismos que permite la interpretacion de datos para observar analizar el comportamiento y rendimiento de los negocios.

¿Para que se creo?

Para demostrar el amplo alcance que este conjunto de tecnologias pueden llegar a ofrecer par ala visualizacion de reportes, ya que no esta limitado a observar el rendimiento de los negocios, si no, analizar ademas el comportamiento de diversas areas o sectores ya sea publicos o privados donde se requiera realizar algun monitoreo o consulta de datos.

Objetivo

Poner al alcance de la mano la visualizacion de reportes graficos con informacion en tiempo real, sin iportar la hora ni el lugar donde se requiera hacer una consulta..

Estrategias

Mediante el uso de dispositivos moviles equipados con Android 2.1 o superior, tecnologias que auxilian el proceso de renderizacion de graficas y conexion a internet para obtener datos.
